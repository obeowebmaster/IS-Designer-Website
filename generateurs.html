Grâce à la technologie open-source Acceleo, il est possible de paramétrer entièrement la génération de code à partir de vos modèles.

Basé sur un principe de template, Acceleo permet d'exploiter n'importe qu'elle partie du modèle pour générer automatiquement des fichiers :

  • dans tout type de langage (Java, PHP, XML, HTML, etc…),
  • en intégrant les frameworks de votre choix,
  • basé sur vos bonnes pratiques de codage,
  • tout en conservant le code ajouté manuellement (entre les balises prévues à cet effet).

Le générateur PACMAN est un projet open-source (licence Apache ASL v2) du Ministère des Armées de la France.

Il permet de générer jusqu'à 80% du code d'une application s'appuyant sur SPI4J, un framework qui fournit une API simple pour les applications JavaEE (d'où son nom : Simple Programming Interface for JavaEE):

  • Architecture SOA basée sur des services métiers, séparés des objets métiers (DTOs et Entities) portant l'état des données
  • Architecture en couches (business, matching, persistence, ui) et N-tiers physiques (client, serveur d'applications, base de données relationnelle)
  • API générée volontairement simple, promeut les bonnes pratiques de développement pour les applications de gestion
